1. 文件说明
======================================================================

该目录为开发包安装程序脚本文件目录：

PWinCVSTrac_Install.nsi	- CVSTracNT 中文版安装程序脚本
                          （用文本编辑器编辑）

制作安装程序时，请先修改 .nsi 文件，更新版本号及新增加的文件和其它内
容，再使用编译工具生成安装程序。


======================================================================
2. 编译工具
======================================================================

安装脚本使用以下工具编译生成安装程序：

NSIS V2.0 即 "Nullsoft Scriptable Installation System"。

该工具由开发 Winamp 的 Nullsoft 开发，是一个轻量级的使用脚本语言的安装
程序制作工具，支持宏定义、回调函数等功能。

由于该工具不同版本之间的兼容性不太好，编译时请使用 V2.0 版本，更低版
本（或再高版本）都可能不能正常编译。
